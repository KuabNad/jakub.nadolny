# Academic Website

A simple responsive academic website built with plain HTML and CSS. It includes Home, Research, Publications, Teaching, and Contact sections.

## Customize

Edit `index.html` to replace the placeholder name, department, biography, publications, courses, email address, and profile links.

## Publish With GitHub Pages

1. Create a new GitHub repository.
2. Add `index.html`, `styles.css`, and this `README.md` to the repository root.
3. In GitHub, open **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and the `/root` folder, then save.

Your site will be available at:

```text
https://YOUR-USERNAME.github.io/YOUR-REPOSITORY/
```
